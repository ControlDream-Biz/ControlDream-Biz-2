# PWA 安装指南

## 📱 什么是 PWA？

PWA（Progressive Web App）是一种渐进式Web应用，可以像原生应用一样安装到您的设备上，享受更好的使用体验。

## 🎯 为什么推荐 PWA？

相比 Electron（桌面应用），PWA 有以下优势：

| 特性 | Electron | PWA |
|------|----------|-----|
| 下载大小 | ~150MB | 无需下载 |
| 安装过程 | 需要安装程序 | 一键安装 |
| 更新方式 | 需要重新下载 | 自动更新 |
| 跨平台 | 需要打包3个版本 | 一个版本 |
| 离线访问 | 支持 | 支持 |
| 桌面快捷方式 | 支持 | 支持 |

## 🔧 PWA 功能说明

### 1. 自动安装提示
- 首次访问管理后台时，系统会自动弹出安装提示
- 如果不希望每次都提示，可以选择"不再提示"

### 2. 离线访问
- PWA 支持离线访问核心功能
- 离线时可以查看已缓存的内容
- 重新联网后自动同步数据

### 3. 桌面快捷方式
- 安装后会在桌面创建快捷方式
- 双击即可打开，无需浏览器
- 独立窗口运行，体验更佳

### 4. 自动更新
- PWA 会自动检查更新
- 无需手动下载新版本
- 后台静默更新，不影响使用

## ❓ 常见问题

### Q1: 为什么看不到安装提示？

**A:** 可能的原因：
1. 浏览器不支持 PWA（请使用 Chrome 或 Edge）
2. 已经安装过了
3. 之前选择了"不再提示"

**解决方法：**
- 清除浏览器缓存后重新访问
- 或手动创建快捷方式（Chrome → 菜单 → 更多工具 → 创建快捷方式）

### Q2: PWA 安全吗？

**A:** PWA 非常安全：
- 使用 HTTPS 加密传输
- 遵循同源策略
- 数据存储在本地沙盒中
- 定期安全更新

### Q3: 可以卸载 PWA 吗？

**A:** 可以卸载，就像卸载普通应用一样：
- Windows: 控制面板 → 程序和功能
- Mac: 应用程序文件夹 → 删除
- iOS: 长按图标 → 删除应用
- Android: 长按图标 → 卸载

### Q4: PWA 会占用很多空间吗？

**A:** 不会，PWA 通常只占用 5-20MB 空间，主要用于缓存。

### Q5: PWA 支持通知吗？

**A:** 是的，PWA 支持推送通知功能（需要用户授权）。

## 💡 技术说明

### PWA 技术栈
- **Manifest**: 应用清单文件（`public/manifest.json`）
- **Service Worker**: 后台服务线程（`public/sw.js`）
- **HTTPS**: 安全传输协议
- **缓存策略**: 网络优先 + 缓存回退
