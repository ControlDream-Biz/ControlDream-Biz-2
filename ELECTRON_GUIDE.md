## 📥 安装指南

### Windows

**安装包安装（推荐）:**
1. 下载 `创梦管理后台-1.0.0-x64.exe` 或 `创梦管理后台-1.0.0-ia32.exe`
2. 双击运行安装程序
3. 按照提示完成安装
4. 桌面会出现"创梦管理后台"快捷方式

**免安装版（便携版）:**
1. 下载 `创梦管理后台-1.0.0-x64.exe` 或 `创梦管理后台-1.0.0-ia32.exe`
2. 解压到任意目录
3. 直接运行 `创梦管理后台.exe`

### macOS

1. 下载 `创梦管理后台-1.0.0.dmg`
2. 双击打开 DMG 文件
3. 将应用拖拽到"应用程序"文件夹
4. 在"应用程序"中找到并运行应用

**注意**: macOS 可能会提示"来自身份不明开发者"，请在"系统设置" → "隐私与安全性"中允许。

### Linux

**AppImage（通用）:**
1. 下载 `创梦管理后台-1.0.0.AppImage`
2. 添加执行权限: `chmod +x 创梦管理后台-1.0.0.AppImage`
3. 运行: `./创梦管理后台-1.0.0.AppImage`

**Debian/Ubuntu:**
1. 下载 `创梦管理后台-1.0.0.deb`
2. 安装: `sudo dpkg -i 创梦管理后台-1.0.0.deb`
3. 修复依赖（如有）: `sudo apt-get install -f`

**Fedora/RHEL:**
1. 下载 `创梦管理后台-1.0.0.rpm`
2. 安装: `sudo rpm -i 创梦管理后台-1.0.0.rpm`

### 开发者模式

2. 点击"关于"
3. 记下版本信息
4. 使用快捷键 `Ctrl+Shift+I` 打开开发者工具

### 数据目录
- **Windows**: `%APPDATA%\chuangmeng-admin`

## ❓ 常见问题

### Q1: 应用无法启动？
**A**:
- 检查系统版本是否符合要求
- 重新安装应用
- 查看日志文件定位问题

### Q2: 窗口无法显示？
**A**:
- 尝试快捷键 `Ctrl+Shift+H` 显示窗口
- 重启应用
- 检查任务管理器是否有进程残留

### Q3: 功能无法使用？
**A**:
- 检查网络连接
- 尝试清除缓存
- 重新登录管理后台

### Q4: 如何卸载？
**A**:
- **Windows**: 控制面板 → 程序和功能 → 卸载
- **macOS**: 将应用拖到废纸篓
- **Linux**: `sudo dpkg -r chuangmeng-admin` (Debian/Ubuntu)

### Q5: 如何更新？
**A**:
- 应用会自动检查更新
- 也可以在"关于"中手动检查
- 下载最新安装包覆盖安装

### Q6: 数据会丢失吗？
**A**:
- 不会，数据保存在本地数据目录
- 卸载时数据目录会保留
- 如需完全清理，手动删除数据目录
