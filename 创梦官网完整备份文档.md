## 重要说明

### 1. Tailwind CSS 版本注意事项

⚠️ **重要**: 本项目使用 Tailwind CSS 3.3.3（而非最新的4.x版本）

**原因**: Tailwind CSS 4.x 的 lightningcss 原生模块在某些环境无法正确加载，导致500错误。

**修复历史**:
- 原版本: Tailwind CSS 4.1.18
- 降级到: Tailwind CSS 3.3.3
- 移除了 `@import 'tw-animate-css'`（Tailwind 3不支持）

### 2. CSS 兼容性修复

以下组件已修复 Tailwind CSS 4 的兼容性问题：

- `toggle-group.tsx`: `gap-[--spacing(var(--gap))]` → `gap-[var(--gap)]`
- `calendar.tsx`: `[--cell-size:--spacing(8)]` → `[--cell-size:2rem]`
- `sidebar.tsx`: `(--spacing(4))` → `1rem`
- `alert.tsx`: `calc(var(--spacing)*4)` → `1rem`

### 3. 端口配置

- 开发环境端口: **5000**（必须使用此端口）
- 生产环境端口: **5000**（必须使用此端口）
- 禁止使用 9000 端口（系统保留）

### 4. 环境变量

项目使用以下环境变量（由沙箱自动提供）：

- `COZE_WORKSPACE_PATH`: 项目工作目录
- `COZE_PROJECT_DOMAIN_DEFAULT`: 对外访问域名
- `DEPLOY_RUN_PORT`: 服务监听端口
- `COZE_PROJECT_ENV`: 环境类型（DEV/PROD）

**禁止硬编码**这些值，必须在代码中通过环境变量获取。

### 5. 字体渲染优化

项目实现了极致清晰的字体渲染（`fonts.css`）：

```css
* {
  -webkit-font-smoothing: antialiased;
  -moz-osx-font-smoothing: grayscale;
  text-rendering: optimizeLegibility;
}
```

### 6. 动画性能优化

- 移除了强制 120HZ 动画设置
- 浏览器会自动根据显示器刷新率渲染
- 使用 GPU 加速（`transform: translateZ(0)`）
- 优化滚动性能（`overscroll-behavior: none`）

### 7. iOS 16 液态玻璃效果

项目实现了iOS 16风格的液态玻璃UI：

```css
.glass {
  background: rgba(255, 255, 255, 0.78);
  backdrop-filter: blur(32px) saturate(160%) brightness(1.02);
  border: 1px solid rgba(255, 255, 255, 0.72);
  /* 更多效果... */
```

## 常见问题

### Q: 为什么 Tailwind CSS 版本是 3.3.3？

A: Tailwind CSS 4.x 的 lightningcss 原生模块在某些环境无法正确加载，导致500错误。降级到 3.3.3 可以解决此问题。

### Q: 如何添加新页面？

A: 在 `src/app/page.tsx` 中添加新的页面组件，并在 `pages` 数组中注册。

### Q: 如何修改动画效果？

A: 修改 `src/app/globals.css` 中的动画定义，或在组件中使用 Tailwind 的动画类。

### Q: 如何自定义颜色？

A: 修改 `tailwind.config.js` 中的 `theme.extend.colors` 或 `globals.css` 中的 CSS 变量。
