# 背景音乐文件目录

## 需要添加的文件

请将 "永远的朋友" MP3 文件命名为 `forever-friends.mp3` 并放置到此目录中。

## 文件信息

- 文件名：forever-friends.mp3
- 格式：MP3 (audio/mpeg)
- 用途：网站背景音乐
- 播放方式：循环播放

## 注意事项

1. 确保文件名为 `forever-friends.mp3`（小写）
2. 文件大小建议不超过 10MB 以保证加载速度
3. 推荐比特率：128kbps - 192kbps
4. 音乐将自动循环播放

## 替代方案

如果您无法找到此音乐，可以使用任何其他MP3文件重命名为 `forever-friends.mp3`
