# 背景音乐说明

## 音乐文件要求

请将《永远的朋友》MP3 文件放置到此目录，并命名为 `forever-friends.mp3`。

## 完整路径

```
/workspace/projects/public/music/forever-friends.mp3
```

## 文件规格建议

- **格式**: MP3
- **MIME类型**: audio/mpeg
- **文件大小**: 建议 ≤ 10MB（最佳加载速度）
- **比特率**: 128kbps - 192kbps
- **采样率**: 44.1kHz
- **时长**: 不限（建议 3-5 分钟）

## 功能特性

- ✅ 页面加载后自动播放
- ✅ 循环播放
- ✅ 可通过右上角按钮控制播放/暂停
- ✅ 支持音量调节（垂直滑块）
- ✅ 播放时显示暂停图标
- ✅ 暂停时显示播放图标

## 控制按钮位置

- **播放/暂停按钮**: 右上角，距离顶部 96px
- **音量控制**: 播放按钮下方，垂直滑块（仅播放时显示）

## 注意事项

1. **浏览器自动播放限制**: 现代浏览器（Chrome、Safari等）会阻止带有声音的媒体自动播放，除非用户与页面有过交互。如果音乐没有自动播放，这是正常的浏览器行为。

2. **用户体验**: 首次访问时，用户需要点击页面任意位置后，音乐才能开始播放。这是浏览器的隐私保护机制。

3. **移动端**: iOS Safari 和部分 Android 浏览器对自动播放有更严格的限制，需要用户交互才能播放。

4. **文件名**: 必须严格为 `forever-friends.mp3`（全小写）

## 如何添加音乐

### 方法1：直接放置文件
1. 找到 `/workspace/projects/public/music/` 目录
2. 将 `forever-friends.mp3` 文件复制到该目录

### 方法2：使用命令（如果您有文件）
```bash
# 假设您的音乐文件在其他位置
cp /path/to/your/music.mp3 /workspace/projects/public/music/forever-friends.mp3
```

## 替代方案

如果您暂时无法找到《永远的朋友》这首歌，可以使用任何其他MP3文件，重命名为 `forever-friends.mp3` 即可。

音乐加载后，页面右上角会出现一个圆形的播放控制按钮。
